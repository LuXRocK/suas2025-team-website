21.04.2025 - In lab tests of software solution

Since we're tasked with object detection and classification we started on some software approach of that topic. We couldn't get a hand on all of the objects from the object list to train a YOLO model for our needs, so we decided to use a pre-trained model.

The object in the image above is a umbrella but the YOLO model mistakes it for a sports ball. We realised that even though the classification is wrong, the model does a good job at detecting objects on the ground. We decided to use this model as a image cropper, we take the bounding box of the detected object and crop the image.

After cropping the image we decided to use a large language model runnig locally on the Jetson to classify the object.

The image above shows LLM's reasoning process and final answer. We can see that it can classify the object correctly. We decided to use this approach because we can use the same model for all of the objects on the list.
